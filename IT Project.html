Website production

Let everyone who can't make web pages have their own web pages. Customers who need to make web pages can contact me through this web page to tell me what functions the web page needs and how the style of the web page should be. Record and sort out the needs of customers, and make a plan for web page production. First, design a web page template for customers to check. After confirming that customers are satisfied, start making web pages according to the template. After making the web page, deliver it to the customer. After the customer confirms that there is no problem, it is completed.
